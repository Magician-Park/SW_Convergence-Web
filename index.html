창의공학설계 G조 프로젝트 설명

          # 제작에 쓰인 부품
          • 제품의 프로토타입을 만들 때 주로 쓰이는 브레드보드와 점퍼 케이블, LED 4개(빨강,
              노랑, 초록), RGB LED, 220옴 저항의 부품을 사용하여 제작했습니다.

          # beautifulsoup
          • BeautifulSoup Documentation를 참고, BeautifulSoup의 라이브러리를 크롤링하여
             날씨 정보와 미세먼지 정보를 LED로 나타내는 파이썬 코드문을 작성했습니다.
          • beautifulsoup 라이브러리는 HTML 을 XML 형식의 파이썬 객체로 변환하여
              웹 페이지의 정보를 탐색하는데 도움을 주는 라이브러리 입니다.
