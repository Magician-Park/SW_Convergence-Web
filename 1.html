날씨 정보(XML)를 LED로 표현하기

# 개요

날씨 정보를 제공하는 기상청 사이트를 통해 오늘의 날씨정보를 알아본다.
날씨정보에 따라 LED에 불이 들어올 수 있도록 파이썬 코드를 작성한다.
구매한 라즈베리파이 부품을 이용하여 회로도를 구성하고 작성한 코드를 실행한다.
날씨 정보를 1:맑음(하얀 LED), 2:구름 조금(초록 LED), 3:구름 많음(노란 LED), 4:흐림(빨간 LED)로 표시한다.
오늘 날씨 정보에 따라 LED가 동작하는지 확인한다.


# 과정



  

    
  




기상청 사이트에서 제공하는 XML 코드를 크롤링 해와 파이썬 코드를 작성하였다.
위 회로도대로 케이블을 연결한 다음 코드를 입력하였더니 정상적으로 결과가 도출되었다.


# 결과물



  

    
  




코드를 작성 후 실행 시킨 결과 2021/11/26일 금요일 오후 7시를 기준으로 전라북도 날씨 정보는
4:흐림(빨간 LED)이 나온 것을 통해 날씨가 흐리다는 결론을 도출
